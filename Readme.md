
##What can you do with it?
  + Search for active Github profiles


## Technologies Used:

  + HTML
  + CSS
  + Javascript
  + Sinatra
  + Github API
  
### How to set it up
```sh
git clone git@github.com:nikeshashar/GithubAPI.git
cd GithubAPI
bundle
```

### How to run it
```sh
cd GithubAPI
rackup
```

open your browser and go to [localhost:4567](http://localhost:4567)
